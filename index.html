Installation & Deployment

Local Setup

npm install – installs all dependencies from package.json into node_modules.

npm run build – runs the build script to compile Tailwind CSS, process assets, and output a production-ready site (usually in dist/ or public/).

Netlify Deployment

Connect your GitHub repo; Netlify automatically runs npm install and npm run build on the server.

Set Build command to npm run build and Publish directory to your build folder (e.g., dist/).

Using Images

Place images in public/ and reference them from root: or Tailwind: bg-[url('/images/hero.jpg')]. Ensures images are copied to the build output and work both locally and on Netlify.
